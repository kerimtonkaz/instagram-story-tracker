### Introduction

Instagram Story Tracker ("Extension") values user privacy. This privacy policy explains what data the extension collects, how it uses it, and how it protects it.

### Data Collected

The extension collects and stores the following data **locally only (on your device)**:

1. **Instagram Username:** Your own username entered in settings
2. **Story Viewer Data:** Names of users who view your Instagram stories
3. **Watch List:** Usernames you want to receive notifications for
4. **App Settings:** Check interval, notification preferences, theme, and language selection

### Data Usage

- All data is stored **only on your device** (in Chrome's local storage)
- **No data is sent to external servers**
- **No data is shared with third parties**
- Data is used solely to provide the extension's core functionality

### Data Security

- Data is stored using Chrome's secure `chrome.storage.local` API
- Data never leaves your device
- All data is automatically deleted when you uninstall the extension

### Third-Party Services

This extension:
- Does **NOT** use analytics tools
- Does **NOT** use advertising networks
- Does **NOT** send data to external APIs

### Required Permissions

| Permission | Why Required |
|------------|--------------|
| `storage` | To store your settings and story data locally |
| `alarms` | To create timers for periodic story checks |
| `notifications` | To send desktop notifications |
| `tabs` | To manage Instagram tabs |
| `scripting` | To extract story viewer list |
| `offscreen` | To play notification sounds in the background |
| `host_permissions (instagram.com)` | To fetch data only from Instagram |

### User Rights

At any time, you can:
- Delete all your data from extension settings
- Remove the extension to clear all data
